# C project template

## Needed tools

- git
- gcc
- autotools
- make
- lex/yacc
- valgrind
- cppcheck
- astyle _(manual build)_
- fortune

## Build

```text
$ ./bootstrap.sh
$ make
$ make check cppcheck
$ make help
```
